Teamkill Homefront plugin for Big Brother Bot (www.bigbrotherbot.net)
=====================================================================

By Courgette


Description
-----------

This plugin kill any player kill a team mate for Homefront



Installation
------------

 * copy tkhf.py into b3/extplugins
 * update your main b3 config file with :

<plugin name="tkhf" />
